# GOES G detector

A GitHub Actions starter project that:
- downloads the latest 7-day GOES magnetometer data from SWPC
- resolves GOES-East and GOES-West from SWPC metadata
- runs a simple rule-based G detector
- outputs a plot and candidate CSV

## Run
Go to Actions -> GOES G Detector -> Run workflow

## Output
Download the artifact:
